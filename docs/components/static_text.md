# Static Text Component

Example JSON:

```
"StaticText": {
    "text": "Winbar!",
    "styles": { ... }
}
```

## Fields

| Key      | Description                                                         |
| -------- | ------------------------------------------------------------------- |
| `text`   | The text to display                                                 |
| `styles` | The styles to apply to the component. See [styling](../styling.md). |
